Radial Basis Function

There are many different types of Kernel. The kernel used in the Hard Margin SVM and Soft Margin SVM examples from earlier are known as Linear Kernels since they didn't require any projections. The kernel used in the projection example is a type of Polynomial Kernel. There are many types of kernel that can be used with Support Vector Machines.
